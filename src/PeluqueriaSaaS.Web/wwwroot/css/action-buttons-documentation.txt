CLASES DISPONIBLES:
------------------
- btn-view: Botón azul para "Ver" (min-width: 90px)
- btn-edit: Botón amarillo para "Editar" (min-width: 90px)
- btn-delete: Botón rojo para "Eliminar" (min-width: 90px)
- btn-create: Botón verde para "Crear" (min-width: 120px)
- btn-back: Botón gris para "Volver/Cancelar" (min-width: 80px)

RESPONSIVE:
-----------
En móviles los botones se centran automáticamente y reducen tamaño.
